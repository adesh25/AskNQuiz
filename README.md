# AskNQuiz

AskNQuiz is a full-stack Online Quiz and Doubt Solving Platform developed using Java Spring Boot, React JS, MySQL, and C# Logger Service.

## Features

### Student Module

* Student Registration
* Student Login
* Attempt Quizzes
* View Quiz Results
* Ask Doubts
* View Doubt Replies

### Teacher Module

* Teacher Login
* Create Quiz
* Add Questions
* Release Quiz
* Reply to Student Doubts
* View Student Results
* Post Notices

### Admin Module

* Admin Login
* Add Teachers
* View Students
* View Teachers
* Delete Students
* Delete Teachers

### Notice Module

* Post Notice
* View Notice

## Technologies Used
